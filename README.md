# vcluster-auto-nodes-pod

A vCluster Platform Terraform `NodeProvider` ([docs](https://www.vcluster.com/docs/platform/administer/node-providers/terraform)) that uses pods as [auto nodes](https://www.vcluster.com/docs/vcluster/deploy/worker-nodes/private-nodes/auto-nodes/) for a vCluster.

```
kubectl apply -f https://raw.githubusercontent.com/FabianKramm/vcluster-tf-example-pod/refs/heads/main/provider.yaml
```
